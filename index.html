About Me

I am a Spanish guy who is currently in his final year of Computer Science for Games at Sheffield Hallam University. Before this final year, I developed Avalo Legends (PC, Steam) with a students team while studying at ESAT Valencia.

During the years of learning, I have gained experience in software development for both PCs and the current generation of consoles. This experience has been acquired in several disciplines although I am more interested in engine programming, graphics and animations. I am an active person so I like to be always doing projects in which to improve my skills as a software developer, teamwork and improve as a person, so I take the projects with creativity, proactivity and enthusiasm.

Currently, as a final year project, I am developing a GPU accelerated animations library, which allows reproducing skeletal mesh animations by GPU or CPU according to the user´s preference and edit them.

Avalo Legends

First Person Melee Game in Unreal Engine 4• 10 / 2018 - 7 / 2019

Avalo Legends is a first-person melee multiplayer game made in Unreal Engine 4 and published in Steam. My main contributions to this game development are:

  • Full implementation of the multiplayer system (split-screen system and input management).
  • Design of class hierarchy and component system by character and implementation of some characters’ ability (e.g. characters' dash).
  • Implementation of some post-processing visual effects (e.g. outline of character).
  • Integration of FMOD plugin for the implementation of the game’s audio.
  • Full game audio system implementation and integration.
